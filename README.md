## How to install
1. Clone the repo: `git clone https://github.com/kurtitamas85/testing_gentoo`
2. Cd in script directiry: `cd testing_gentoo/scripts`
3. Make the script executable: `chmod +x gentoo-install.sh`
4. Run the script: `sudo ./gentoo-install.sh`
5. Refer to the [Manual Guide](docs/guide.md) for full details.
