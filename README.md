Gentoo LiveGUI Automation Project

This repository contains the automation scripts and documentation for a streamlined Gentoo installation process.

Structure

/scripts: Contains the main gentoo-install.sh TUI script.

/docs: Contains the markdown source for the Visual Guide.

Quick Start

To run the automated installer:

git clone [https://github.com/kurtitamas85/testing_gentoo.git](https://github.com/kurtitamas85/testing_gentoo.git)
cd testing_gentoo/scripts
chmod +x gentoo-install.sh
sudo ./gentoo-install.sh
